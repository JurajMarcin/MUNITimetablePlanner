How to import timetable from IS?

1. Open your timetable: https://is.muni.cz/auth/rozvrh/zobraz/muj

2. Select 'Display Options'

3. Switch to 'Programmer' display format and check 'include teachers' names' and 'include all the seminar groups of the courses displayed' and click 'Show'

4. Save the result file and import it

How to use Timetable Planner?

You can change the priority of seminars (lower the number means higher priority). Overlaping seminars are ordered by priority. Lectures have the highest priority by default.

You can also hide lesson, you don't want to see. Hidden lessons can be displayed using show all checkbox in the page header.

Colour of the lesson is determined by its priority.
